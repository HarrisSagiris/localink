How Localink Works

1

Install Localink

Download and install Localink on your devices. Available for all major platforms.

npm install -g localink
2

Create Your Network

Initialize your secure network with a single command.

localink init --network MySecureNetwork
3

Connect & Go

Your devices are now connected in a secure mesh network. Access them from anywhere.

localink connect
